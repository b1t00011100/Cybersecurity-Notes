# NETWORK

- A Group of two or more computers that share services and interacting in some manner is known as **Computer network.**
- Links connecting the Computers in a network are called Communication Channels.
- A Node can be a device which is able to send or receive data generated by other nodes on the network , For ex : a computer , printer.
- A Physical pathway is known as **Transmission Medium.**
- A Computer which is meant for the purpose to host shared hard drives is called a **File Server.**
